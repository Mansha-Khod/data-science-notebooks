#  Data Science Learning Lab

A collection of focused notebooks and scripts covering core data science concepts — visualisation, data mining, machine learning fundamentals, and exploratory analysis.

These are hands-on practice pieces built alongside coursework at Mount Carmel College, Bangalore.

---

## Contents

| File | Topic | Description |
|------|-------|-------------|
| `Tips.ipynb` | EDA | End-to-end exploratory analysis on the Tips dataset — tipping patterns across bill size, party size, and time |
| `Matplotlib_Notebook.ipynb` | Visualisation | Core Matplotlib charts: line plots, bar charts, scatter plots, subplots |
| `Seaborn.ipynb` | Visualisation | Statistical visualisation with Seaborn — heatmaps, violin plots, pair plots |
| `data_cleaning.ipynb` | Data Wrangling | Handling missing values, duplicates, type casting, and outlier detection |
| `desicion_tree.ipynb` | ML | Decision Tree classifier — training, visualisation, and evaluation |
| `FPtree.ipynb` | Data Mining | FP-Tree algorithm for frequent pattern mining and association rules |
| `Data Cube Construction and OLAP Operations.ipynb` | Data Warehousing | Building data cubes and applying OLAP operations (slice, dice, roll-up, drill-down) |
| `Hangman game.py` | Python Practice | Command-line Hangman — string manipulation and game logic |

---

## Tech Stack

Python · Pandas · NumPy · Matplotlib · Seaborn · Scikit-learn · Jupyter Notebook
